### new()

Creates a new dropout block object.

```

Dropout.new({dropoutRate: number}): DropoutBlockObject

```

#### Parameters:

* dropoutRate: The rate at which the input values are converted to zero.

#### Returns:

* DropoutBlock: The generated dropout block object.
